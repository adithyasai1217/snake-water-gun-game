# snake-water-gun-game
Console-based Snake Water Gun game built in C using random number generation and conditional logic.

A simple command-line implementation of the classic Snake Water Gun game written in C.

## Features
- Randomized computer moves
- Input validation
- Win, lose, and draw detection

## Getting Started

### Compile
gcc snake_water_gun.c -o game

### Run
./game

## Technologies Used
- C
- stdio.h
- stdlib.h
- time.h
